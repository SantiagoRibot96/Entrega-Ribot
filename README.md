# Base TodoParrillas
Todo Parrillas es una PyMe real cuyo propósito es el de fabricar parrillas de chapa. Esta empresa se dedica a la transformación desde la materia prima (hierro, chapa) hasta la venta final, encargándose también de la entrega del producto.

## Instalacion de la base de datos
Para lograr una instalacion exitosa, se deben correr los scripts de la siguiente manera:
- 01-Creacion: Crea la base de datos y sus tablas
- 02-Insercion: Inserta contenido en las tablas
- 03-Vistas: Crea vistas para facilitar consultas
- 04-Funciones-Procedimientos: Crea funciones y procedimientos
- 05-Triggers: Crea las alertas
- 06-Sentencias: Crea los usuarios con permiso de escritura o lectura
